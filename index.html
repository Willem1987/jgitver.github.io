jgitver

Industrialize project versioning using git

Welcome to JGitver.

JGitver is community hosting several projects whose goal is to let developers, devops and other project maintainers to industrialize the way they handle their project versioning.

Please visit each jgitver component page in order to have some details on each.

jgitver library

The library responsible for version calculation ; based on jgit it uses git metadatas, tags & history to produce a set of data including a calculated project version.
Standard users will normally use the library via one of its build plugin:

  • jgitver maven plugin
  • gradle jgitver plugin

jgitver maven plugin

maven core extension allowing to define maven POM version using jgitver.
Visit the github project page for details: jgitver-maven-plugin

jgitver gradle plugin

gradle plugin allowing to define build.gradle version using jgitver.
Visit the github project page for details: gradle-jgitver-plugin

jgitver configuration xml-schemas

A list of xml schemas related to jgtiver projects (configuration, ...) is published & maintained here
